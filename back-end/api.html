Sign up

Request

Key Value
Endpoint http://localhost:3100/api/user/signup
Method POST
Example body 
{
  "username":"Dimitrina",
  "email":"dimitrina@mail.com",
  "password": "12345678"
}
Constraints
  • username - string, required, 3-40 characters long
  • email - string, email, required
  • password - string, required, 8 characters min length

Response

Key Value
Response-type JSON
Example body

  "status": "success",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ",

    "user": {

      "username": "Dimitrina",
      "email": "dimitrina@mail.com",
      "__v": 0
    }
  }
}
